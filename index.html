call apply bind

call, apply,bind are use to bowrowing function.

1. call is take argument seperat.

        syn:- 
        const names={
            firstname:'Meghana',
            lastname:'Thul',
            fullname:function(no){
                console.log(this.firstname+ this.lastname+' my faverate no is ' + no)
            }
        }
        names.fullname(9)
        
        
        
        const name1={
            firstname:'aaaa',
            lastname:'bbbnb',
             
        }
         names.fullname.call(name1,9)

2.apply is take arument in arrey formate.

        syn:- 
        const names={
            firstname:'Meghana',
            lastname:'Thul',
            fullname:function(no){
                console.log(this.firstname+ this.lastname+' my faverate no is ' + no)
            }
        }
        names.fullname(9)
        
        
        
        const name1={
            firstname:'aaaa',
            lastname:'bbbnb',
             
        }
         names.fullname.apply(name1,[9])
